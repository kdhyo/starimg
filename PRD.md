# 이미지 월드컵

- imgs에 있는 사진을 가지고, 월드컵게임과 같은 웹사이트를 구현.
- root 페이지에는 중앙에 이름(별명) 입력과 5 or 10 중 하나를 선택하고 엔터
- 다음 페이지부터 imgs의 이미지를 이름순으로 정렬해서 root에서 선택한 숫자의 이미지만큼 보여주기
- 사용자가 원하는 이미지를 갯수 상관 없이 클릭하여 다음으로 넘어가기
- 모든 이미지가 다 선택되고 루프를 돌았으면, 선택된 이미지는 별 1개
- 별이 5개 선택된 이미지들이 있을 때까지 혹은 1개 이미지만 선택이 됐을때까지 반복
- 다 끝난 후, 처음 선택한 이름의 결과 라고 하면서 별 5개 이미지 목록, 별 4개 이미지 목록 순으로 페이지에 노출
- 별 목록 별로 이미지를 저장할 수 있도록 구현
- 맨 입력한 이름과 별 목록을 서버에 저장

체크리스트
- 서버와 클라이언트를 분리하여 구현하여 로컬이 아닌 외부에서 다운로드도 가능하도록 구현.
- 모든 이미지를 한 번에 로드하면 너무 많기 때문에, 적절히 로드 (그렇다고 선택할 때마다 로드하면 서버에 부하가 많지 않을까 싶기 때문에 성능을 고려한 개발 필수)
